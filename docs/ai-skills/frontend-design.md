# Skill — frontend design (adaptada a GymTrack)

Objetivo:
diseñar interfaces rápidas, claras y móviles

Principios:

- minimizar fricción al usuario
- priorizar acciones principales
- evitar sobrecarga visual
- jerarquía clara
- feedback inmediato

En GymTrack:

- inputs rápidos (ej: 60x8)
- botones grandes (➕ ➖ ✓)
- uso con una mano
- navegación simple
- evitar modales innecesarios

Componentes:

- pequeños
- reutilizables
- props simples

Evitar:

- layouts complejos
- formularios largos
- múltiples pasos innecesarios

Prioridad máxima:
velocidad al registrar entrenamientos
